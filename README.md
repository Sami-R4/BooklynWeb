# Booklyn - Online Library System

Booklyn is a mordern, user-friendly **online library** where readers can explore, buy annd download eBooks(PDFs). Built with **PHP (MySQL)** on the backend and **HTML/CSS/Bootstrap** on the front end. Booklyn focuses on a clean UI , smooth broswing, and a delightfull reading experience.

---

## Why Booklyn
-**Community Engagement** -Discussions, clubs, and shared reading momments.
-**Personalized Recomendations** -Smart suggestions based tastes.
-**Gamification/Achivements** -Badges, milestones, and challanges to keep you reading.

---

## Tech Stack
**Frontend:** HTML, CSS, Bootstrap, Font Awesome
**Backend:** PHP 
**Database:** MySQL
